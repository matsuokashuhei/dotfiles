# Shared Guidelines

## Target Audience
Japanese English learners.

## Input
Text is provided via `$ARGUMENTS`.

## Technical Terms
Keep technical terms as-is (e.g., API, async, Docker, ActiveRecord, Git, CI/CD).

## English Level
Use CEFR B1–B2 level English — clear and concise expressions with common vocabulary. Keep sentences short and grammar straightforward, but allow moderate complexity when needed for accuracy. Avoid verbose or overly academic expressions.
