# translate

Translate Japanese text to English.

## Instructions

1. Translate the Japanese text provided in `$ARGUMENTS` to English.
2. Output ONLY the translated text. No preamble, no explanation, no quotes.
3. Keep technical terms (e.g., API, async, Docker, ActiveRecord) as-is.
4. Preserve the original tone (casual/formal).
5. Use CEFR B1–B2 level English — clear and concise expressions with common vocabulary. Keep sentences short and grammar straightforward, but allow moderate complexity when needed for accuracy. Avoid verbose or overly academic expressions.
